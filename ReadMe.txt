# American Sign Language Image Classification using Transfer Learning with Xception
This project aims to detect American Sign Language (ASL) gestures using Transfer Learning with Xception image classification model.

## Prerequisites

Before you begin, ensure you have met the following requirements:

* You have a Google account to access Google Colab.
* You have downloaded the dataset from https://www.kaggle.com/datasets/grassknoted/asl-alphabet
* Configure dataset paths in Notebook

Libraries and Tools:
* Python
* Jupyter Notebook
* Tesnorflow


## Running the Project

To run this project, follow these steps:

1. Open The notebook and configure dataset path.
2. Run the cells

Google Colab

1. Open the notebook file in Google Colab and configure dataset path.
2. Run the cells in the notebook to start the training process and evaluate the model.
